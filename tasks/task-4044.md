# Task 4044: Grading Rubric Builder

**App Name:** ai-grading-rubric-builder
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that generates detailed grading rubrics. Users input assignment type, criteria, and max points, and the AI creates a comprehensive rubric with performance descriptors for each score level.

## Implementation Details
- Form inputs: assignment type, criteria list, max points, academic level, rubric format
- Output: Detailed rubric table with criteria, weightings, and descriptors for each score level
- UI: Dark gradient with orange-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
