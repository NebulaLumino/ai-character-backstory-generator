# Task 4045: Syllabus Builder

**App Name:** ai-syllabus-builder
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that generates complete course syllabi. Users input course title, description, credit hours, and target students, and the AI creates a full syllabus including policies, learning outcomes, readings, assignments, and grading breakdown.

## Implementation Details
- Form inputs: course title, description, credits, target students, instructor name, semester
- Output: Complete syllabus with policies, learning objectives, weekly schedule, readings, and grading
- UI: Dark gradient with orange-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
