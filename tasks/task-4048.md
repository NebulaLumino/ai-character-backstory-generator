# Task 4048: Study Group Matchmaker

**App Name:** ai-study-group-matchmaker
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that matches students into optimal study groups. Users input subjects, schedule availability, and learning styles, and the AI groups students with complementary strengths and weaknesses.

## Implementation Details
- Form inputs: student profiles (name, subjects, availability slots, learning style, strengths/weaknesses), group size
- Output: Suggested study group configurations with rationale and compatibility scores
- UI: Dark gradient with emerald-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
