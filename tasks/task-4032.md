# Task 4032: Flashcard Generator

**App Name:** ai-flashcard-generator
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that automatically generates flashcards from any text, topic, or study material. The AI creates question-answer pairs with spaced repetition metadata (ease factor, interval) to optimize long-term retention.

## Implementation Details
- Form inputs: topic/text input (textarea), number of cards desired, subject area, difficulty level
- Output: Q&A flashcards with spaced repetition metadata (ease factor, interval, due date)
- UI: Dark gradient with indigo-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
