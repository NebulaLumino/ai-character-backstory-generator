# Task 4046: Peer Review Feedback

**App Name:** ai-peer-review-feedback
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI-powered peer review tool for student papers. Users submit a draft paper and optional rubric, and the AI provides constructive feedback on structure, argumentation, citations, and writing quality.

## Implementation Details
- Form inputs: draft paper text, assignment type, rubric (optional), review focus areas
- Output: Structured feedback on structure, arguments, citations, writing quality, and improvement suggestions
- UI: Dark gradient with emerald-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
