# Task 4036: Academic Writing Assistant

**App Name:** ai-academic-writing-assistant
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that helps students and researchers with academic writing. Users input a topic, target word count, and citation format (APA/MLA/Chicago), and the AI generates a structured outline, key arguments, and citation suggestions.

## Implementation Details
- Form inputs: topic, target word count, citation format (APA/MLA/Chicago), academic level
- Output: Structured outline, key points, suggested citations, and writing guidance
- UI: Dark gradient with cyan-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
