# Task 4040: Bibliography Generator

**App Name:** ai-bibliography-generator
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI-powered citation and bibliography generator. Users input source URLs, titles, or author information, and the AI generates properly formatted citations and a complete bibliography in APA, MLA, Chicago, or Harvard format.

## Implementation Details
- Form inputs: source URLs, titles, or author information (one per line), citation format
- Output: Properly formatted in-text citations and a complete bibliography
- UI: Dark gradient with cyan-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
