# Task 4031: Course Outline Generator

**App Name:** ai-course-outline-generator
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI-powered tool that generates structured course curricula and lesson plans. Users input subject, target audience, and course duration, and the AI creates a comprehensive week-by-week curriculum with learning objectives, topics, and activities.

## Implementation Details
- Form inputs: subject, target audience (e.g., undergraduate/graduate/beginner/professional), course duration (weeks), hours per week, prerequisites
- Output: Week-by-week curriculum with learning objectives, topics, readings, and activities
- UI: Dark gradient with indigo-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
