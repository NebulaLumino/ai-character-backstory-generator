# Task 4038: Language Vocabulary Builder

**App Name:** ai-language-vocabulary-builder
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI-powered contextual vocabulary builder that extracts and teaches vocabulary from any text. Users input text or specify a language and target level, and the AI extracts words with definitions, example sentences, etymology, and usage context.

## Implementation Details
- Form inputs: text or topic, target language, proficiency level (A1-C2 CEFR), number of words
- Output: Vocabulary list with definitions, example sentences, etymology, and mnemonic hints
- UI: Dark gradient with cyan-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
