# Task 4049: Concept Explanation Visualizer

**App Name:** ai-concept-explanation-viz
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that explains concepts using analogies, diagrams (described in text), and real-world examples. Users input any concept or term, and the AI provides a multi-layered explanation suitable for the user's level.

## Implementation Details
- Form inputs: concept or term to explain, audience level (beginner/intermediate/advanced), explanation style preference
- Output: Multi-layered explanation with analogies, text-based diagrams, and real-world examples
- UI: Dark gradient with emerald-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
