# Task 4039: Lecture Notes Summarizer

**App Name:** ai-lecture-notes-summarizer
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that summarizes lecture recordings or notes into structured summaries. Users paste transcripts or notes, and the AI creates a structured summary with key concepts, timestamps, and main takeaways.

## Implementation Details
- Form inputs: lecture transcript or notes (textarea), lecture title, course name, duration
- Output: Structured summary with key concepts, timestamps, and bullet-point takeaways
- UI: Dark gradient with cyan-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
