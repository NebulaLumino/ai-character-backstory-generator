# Task 4042: Abstract Generator

**App Name:** ai-abstract-generator
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that generates structured research paper abstracts. Users input paper title, methodology, key findings, and conclusion, and the AI produces a well-structured 150-300 word abstract suitable for journal submission.

## Implementation Details
- Form inputs: paper title, research methodology, key findings, conclusion, target journal
- Output: Structured 150-300 word abstract with background, methods, results, and conclusions
- UI: Dark gradient with orange-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
