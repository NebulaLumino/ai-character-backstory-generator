# Task 4043: Learning Gap Diagnoser

**App Name:** ai-learning-gap-diagnoser
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that identifies specific knowledge gaps in students. Users input a subject and the student's answers or work samples, and the AI diagnoses weak areas with targeted learning resources and next-step recommendations.

## Implementation Details
- Form inputs: subject area, student's answers or work description, assessment type
- Output: Gap analysis report with identified weak areas, explanations, and recommended resources
- UI: Dark gradient with orange-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
