# Task 4041: Academic Integrity Checker

**App Name:** ai-academic-integrity-checker
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that analyzes manuscripts and academic texts for citation gaps and potential plagiarism issues. The AI flags missing attributions, suggests proper citations, and provides an integrity score with actionable recommendations.

## Implementation Details
- Form inputs: manuscript text (textarea), subject area, citation style
- Output: Integrity report with flagged sections, citation suggestions, and a summary score
- UI: Dark gradient with orange-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
