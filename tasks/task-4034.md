# Task 4034: Quiz Generator

**App Name:** ai-quiz-generator
**Priority:** HIGH
**Theme:** AI × Education, Learning, Research & Academic Tools

## App Description
An AI tool that generates customized quizzes from any topic or content. Users specify the topic, difficulty level, and question types, and the AI produces multiple choice, true/false, and short answer questions with correct answers.

## Implementation Details
- Form inputs: topic, difficulty (easy/medium/hard), question types (multiple choice/true-false/short answer), number of questions
- Output: Formatted quiz with questions, answer options, and correct answers
- UI: Dark gradient with indigo-500 accent
- API: DeepSeek via /api/generate route
- Build: `npx next build` must pass
